# 交易知识升级记录

更新时间：2026-03-13

这次升级不再只围绕传统技术指标，而是把三类更接近实盘的知识接进应用：

1. A股制度约束
- 股票交易按 `T+1` 执行，买入当日不能卖出。
- `9:15-9:25` 为开盘集合竞价，其中 `9:20-9:25` 不能撤单。
- `14:57-15:00` 为收盘集合竞价，收盘阶段同样存在不能撤单的约束。
- 主板普通股票通常按 `10%` 涨跌幅限制，创业板和科创板按 `20%`，风险警示股票通常按 `5%`。

2. 时段化消息理解
- 同样是利好或利空，`盘中消息` 和 `盘后/隔夜消息` 对第二天开盘行为的影响并不完全一样。
- 当前应用把消息拆成 `盘中情绪`、`隔夜情绪` 和 `次日参考情绪` 三层，避免只看一个总分。

3. 分时结构与执行纪律
- 开盘半小时量能在全天中的占比，可以帮助区分“开盘先手很强”还是“需要等午前/午后再确认”。
- 执行上不鼓励在集合竞价不可撤单窗口追单，而是更强调 `9:35-10:15` 的均价线确认和关键价位确认。
